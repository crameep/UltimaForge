# server-data/

This directory is gitignored. It holds server owner data that must never be committed.

## Structure

```
server-data/
  client/      - Drop your game client files here (ClassicUO.exe, DLLs, Files/, Data/, etc.)
  publish/     - Generated by "Publish" (Option E). This is what gets served / deployed.
  keys/        - Ed25519 signing keys for game updates + Tauri updater keys + VPS deploy key
  deploy.json  - VPS connection config (host, user, domain). Generated by "Setup VPS" (Option H).
```

## Workflows

**Publish a game update:**
1. Drop new game files into `server-data/client/` (overwrite old files)
2. Run Option E from ultimaforge.bat

**Deploy to VPS:**
1. Run Option H (Setup VPS) - first time only
2. Run Option E to publish
3. Run Option I to deploy

**Self-host locally:**
Run Option 5 (Start Test Server) - this serves `app/test-updates/` for dev.
For production local hosting, point host-server at `server-data/publish/`.
